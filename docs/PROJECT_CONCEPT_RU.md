# Концепция проекта OfficeChat

OfficeChat - открытый self-hosted корпоративный чат для локальных сетей, закрытых контуров и организаций, которым нужен контроль над данными.

Основной сценарий MVP - работа внутри LAN или offline-среды без обязательной зависимости от внешних сервисов. При этом архитектура должна позволять позже безопасно публиковать систему в интернет через reverse proxy, TLS, расширенную аутентификацию и политики доступа.

## Принципы

- Простое локальное развертывание через Docker Compose.
- Администратор создает пользователей по умолчанию.
- Саморегистрация отключена по умолчанию.
- Хранение файлов в локальном Docker volume на этапе MVP.
- Подготовка архитектуры к ботам, интеграциям и AI-провайдерам.
- Предпочтение permissive open-source зависимостям.

## Не входит в первый scaffold

- Полноценный чат.
- Аутентификация.
- WebSocket-сообщения.
- LDAP/AD.
- Production nginx-конфигурация.
