# OfficeChat 0.1.0-rc2

OfficeChat 0.1.0-rc2 - release candidate для установки через Docker Compose bundle и GHCR images.

## Что входит

- Auth, сессии и audit log.
- Пользователи, роли, гранулярные права.
- Группы, direct messages, discussions.
- WebSocket realtime, presence, typing, unread counters, read receipts.
- Файлы, multiple attachments, paste/drag-and-drop, inline image previews.
- Mentions, replies, reactions, pinned messages, search.
- Broadcast announcements, Notification Center, browser notifications.
- Calendar Events v0.1 и `calendar-worker`.
- Branding/About, health/readiness metadata.
- Release packaging: GHCR image strategy, release Compose, install/update/rollback/uninstall scripts, bundle generator, reverse proxy examples.

## Образы

- `ghcr.io/fedorovdo/officechat-backend:0.1.0-rc2`
- `ghcr.io/fedorovdo/officechat-frontend:0.1.0-rc2`
- `sha-<short_git_sha>` immutable tags.
- `rc` optional moving tag.

## Известные ограничения

- RC не является стабильным production-релизом.
- Нет LDAP/AD.
- Нет S3/object storage и antivirus scanning.
- Нет recurring calendar, RSVP, ICS, CalDAV, Google/Outlook integration.
- Browser notifications работают только когда OfficeChat открыт в браузере.
- WebSocket manager single-instance; multi-instance потребует Valkey pub/sub.
- `latest` tag не используется как зависимость deployment.

## Ручные действия после проверки

```bash
git tag -a v0.1.0-rc2 -m "OfficeChat 0.1.0-rc2"
git push origin v0.1.0-rc2
```

Публикация GHCR images и GitHub Release выполняются вручную или через workflow после approval.
